# La-Boite-Livre
Application web pour visualiser et suivre l'état des boîtes à livres du quartier.

📚 Gestionnaire de Boîtes à Livres
📋 Présentation du projet

Ce projet a été développé pour faciliter l'accès à la culture dans mon quartier en permettant de visualiser et de suivre l'état des boîtes à livres locales.  L'idée est de savoir en un coup d'œil si une boîte est pleine, vide ou nécessite une attention particulière.


🚀 Fonctionnalités

Visualisation : Affichage de la localisation des boîtes à livres sur une interface interactive. 

Suivi d'état : Indicateur de remplissage des boîtes pour les utilisateurs.

Interface Responsive : Accessible aussi bien sur ordinateur que sur smartphone.

Langages : Python  (ou mentionne le langage utilisé, comme HTML/JS).
